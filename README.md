# Corona Pandemic Quiz

An interactive quiz application to test your knowledge about the virus, measures, and medical background.

## Features

* **Various Question Types**:
  * Yes/No questions
  * Multiple Choice
  * Text Input (e.g., for abbreviations or names)
  * Multiple Selection
* **Player Settings**: Personalization via Name, Age, Gender, and Country.
* **Visual Feedback**: Immediate feedback for correct or incorrect answers.
* **Score System**: Overview of correct and incorrect answers at the end of the game.
* **Joker Question**: Special round with a Joker mechanic (50:50 chance if answered correctly).

## Technologies

* HTML5
* CSS3
* JavaScript (ES6+)
* [Onsen UI](https://onsen.io/) (for mobile app design via Web Components)

## Installation & Usage

1. Download or clone the repository.
2. Open the `index.html` file in a modern web browser (Chrome, Firefox, Safari, Edge).
3. The quiz starts directly in the browser.
